# Install and config

Source: https://github.com/pannguyen2001/capstone_prj_sqlite

## 1. Common config Jest.config.json, tsconfig.test.json
1. Install pnpm globaly
```
npm install -g pnpm
```
2. Install jest
```
pnpm add -D jest ts-jest @types/jest
```

## 2. Required package

- @types/jest
- @types/node
- tsx
- jest
- ts-jest
